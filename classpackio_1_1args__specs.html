Theprocedure

CLass that describes the arguments of the procedure They each have a name and optionally a default value This is a tuple-like class where each element can be constructured from:

  • a string, defining the name of the argument
  • a arg, defining the name of the argument
  • a arg::with_value, defining the name of the argyment and its default value Optionally, accepts args_specs_options as first argument to customize the behavior of argument handling
